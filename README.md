《原神》是由上海米哈游制作发行的一款开放世界冒险游戏，于2017年1月底立项 [29]  ，原初测试于2019年6月21日开启 [1]  ，再临测试于2020年3月19日开启 [2]  ，启程测试于2020年6月11日开启 [3]  ，PC版技术性开放测试于9月15日开启，公测于2020年9月28日开启 [4]  。在数据方面，同在官方服务器的情况下，iOS、PC、Android平台之间的账号数据互通，玩家可以在同一账号下切换设备。
